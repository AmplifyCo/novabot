# Calendar Tool — Usage Guidelines

## Before Scheduling
- ALWAYS check the calendar for conflicts before creating an event
- Check the principal's timezone — display and create events in their local time
- Look for back-to-back meetings and suggest buffer time (15-30 min) when appropriate

## Creating Events
- Include clear titles — "Meeting with John" not just "Meeting"
- Add location or video link if mentioned
- Set appropriate duration: default 30 min for calls, 60 min for meetings
- Add attendees only when explicitly mentioned by the principal

## Conflict Handling
- If there's a conflict, report it clearly: "You already have [X] at that time"
- Suggest alternative times rather than just saying "not available"
- Never silently overwrite or delete existing events

## Privacy
- When someone asks about the principal's availability, say "that time doesn't work" — NEVER reveal what the conflict is or who they're meeting
- Share only general availability windows, not specific event details
