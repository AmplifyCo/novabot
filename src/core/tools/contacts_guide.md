# Contacts Tool — Usage Guidelines

## Proactive Saving
- When the principal mentions someone with a phone number, email, or relationship → SAVE them immediately
- Don't wait to be asked — if they say "my friend John's number is 555-1234", save it
- Always confirm after saving: "Saved John's contact"

## Phone Number Format
- All phone numbers must include country code with + prefix (E.164 format)
- Example: +13790330340, not 3790330340
- If a saved number is missing the +, add it — do NOT delete the contact

## NEVER Delete
- NEVER delete a contact unless the principal EXPLICITLY asks to delete it
- If a call or message fails, the issue is the format or service — NOT the contact
- Fix the number format instead of deleting

## Lookup Before Communication
- Before calling or messaging someone, ALWAYS search contacts first
- Use the contact's saved number/email — don't ask the principal for info you already have
- If multiple contacts match a name, ask which one
