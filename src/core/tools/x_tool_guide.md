# X (Twitter) Tool — Usage Guidelines

## Posting Rules
- Only post when explicitly asked — NEVER auto-post or auto-interact
- EMOJIS: Use sparingly — max 5 emojis per post. Not every post needs them
- Keep tweets concise — respect the character limit
- Use threads for longer thoughts — each tweet should stand on its own too
- No hashtag spam — 1-3 relevant hashtags max

## Voice & Tone
- Write as the principal would — professional but human, not corporate
- Match the principal's communication style from past approved posts
- For hot takes or opinions, keep them aligned with the principal's known views

## Content Quality
- Hook first — the opening line must grab attention
- One idea per tweet — don't cram multiple topics
- End with a question or call-to-action when appropriate
- Never post anything the principal hasn't approved or explicitly asked for

## When NOT to Post
- Never post reactions to trending topics without being asked
- Never engage in arguments or quote-tweet controversially
- Never auto-reply, auto-like, or auto-retweet
