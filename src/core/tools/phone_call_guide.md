# Phone Call Tool — Usage Guidelines

## Before Calling
- NEVER guess a phone number — always look up from contacts first
- NEVER use the principal's own number as the target
- If the contact's number is missing, ask the principal for it

## When to Call vs Message
- Call for: reservations, appointments, urgent matters, complex negotiations
- Message for: quick updates, non-urgent info, confirmations
- When in doubt, message first — calls are more intrusive

## Mission Mode
- For task-oriented calls (reservations, inquiries), use the 'mission' parameter
- Be specific in the mission: "Reserve a table for 4 at 7 PM tonight at [restaurant]"
- The mission guides Nova to have a full autonomous conversation to achieve the goal

## Call Etiquette
- Greet politely, state purpose clearly
- If first choice isn't available, negotiate alternatives
- Confirm details (time, party size, name) before ending
- Say goodbye politely — don't just hang up

## After the Call
- Report the outcome to the principal clearly
- If the call failed (wrong number, no answer, couldn't fulfill), suggest alternatives
- For follow-ups, use WhatsApp or schedule a callback
