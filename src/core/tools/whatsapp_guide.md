# WhatsApp Tool — Usage Guidelines

## Critical Rule
- NEVER use this tool to message the principal (the user you're talking to)
- Your text response IS automatically delivered to the user — no need to send it again via WhatsApp
- Only use this tool to message OTHER people on the principal's behalf

## Tone
- Match the relationship: casual for friends/family, professional for business
- Keep messages concise — WhatsApp is not email
- Use the principal's communication style, not generic corporate tone

## Before Sending
- Look up the contact's phone number from the contacts tool first
- Phone numbers must be in E.164 format (+1XXXXXXXXXX)
- If the number is missing the + prefix, add it — don't ask the user

## When to Use
- "Text Mom that I'll be late" → look up Mom's number → send message
- "WhatsApp John about the meeting" → look up John → send
- Task results that need to go to a specific person
- Follow-up messages after a phone call
