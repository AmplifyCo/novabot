# x402 Tool — Usage Guidelines

## Safety
- Always check the payment amount before paying — never exceed max_amount cap
- Default max_amount is $1 USDC — only raise if the owner explicitly approves
- If a resource costs more than expected, report the price and ask before paying

## When to Use
- Use "check" first to probe whether a URL is 402-gated before paying
- Use "pay_and_fetch" only when the owner wants the gated content and approves the cost
- Never pay for resources speculatively — always have a clear reason

## Communication
- Tell the owner the price before paying: "This resource costs $0.05 USDC. Should I proceed?"
- After payment, report what was received and the transaction details
- If payment fails, explain why (insufficient funds, network error, etc.)
