# ERC-8004 Tool — Usage Guidelines

## Identity Management
- Nova's on-chain identity is registered via ERC-8004 on Base
- The registered handle is Nova's permanent on-chain name — choose carefully
- Registration is IRREVERSIBLE and costs gas — always confirm with the owner first

## When to Use
- "What's my on-chain identity?" → my_identity
- "Look up [handle]" → resolve
- "Verify [address]" → verify
- Registration should only happen once, on owner's explicit request

## Communication
- When resolving identities, present results clearly: handle, address, metadata
- For verification, give a clear yes/no with supporting details
